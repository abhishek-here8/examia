# EXAMIA 🎓

**EXAMIA** ek simple aur user-friendly online platform hai jahan students apne exams ki taiyari behtar tareeke se kar sakte hain. Is website ka main maqsad students ko purane saalo ke papers (PYQs) solve karne ka ek real-time experience dena hai.

---

## 🚀 Main Features

- **User Registration & Login:** Naye users apna account bana sakte hain aur secure login ke zariye platform ka access pa sakte hain.

- **Profile Management:** Login karne ke baad aap apni profile set kar sakte hain, jahan aap apni details aur apni progress track kar sakte hain.

- **Previous Year Questions (PYQs):** Is platform ka sabse bada feature hai PYQs attempt karna. Students alag-alag subjects aur exams ke purane papers ko test format mein solve kar sakte hain.

- **Instant Performance Check:** PYQ attempt karne ke baad aap dekh sakte hain ki aapne kitne sawal sahi kiye aur kahan sudhaar ki zaroorat hai.

---

## 📖 Kaise Use Karein?

1. **Sign Up / Login:** Sabse pehle apna account banayein ya login karein.
2. **Profile Setup:** Apni profile mein apni basic details fill karein taaki aapka data save rahe.
3. **Select Exam:** Dashboard par jaakar wo exam select karein jiske PYQs aap attempt karna chahte hain.
4. **Start Practice:** "Attempt PYQ" par click karein aur apni practice shuru karein.
